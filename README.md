# Airbnb Listings

Simple Airbnb listings page for class.

## What it does

- Loads data from `airbnb_sf_listings_500.json`
- Uses JavaScript `fetch` and `await`
- Shows the first 50 listings
- Shows listing name, description, amenities, host, price, and image
- Extra feature: simple search box and price label
